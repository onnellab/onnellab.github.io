Melivra - 오프라인 음악 플레이어

간단한 설명:
여러 로컬 오디오 파일을 정리하고 듣는 오프라인 음악 플레이어

랜딩 부제:
로컬 오디오 라이브러리 플레이어

자세한 설명:
Melivra는 여러 오디오 포맷의 로컬 음악 컬렉션을 위한 오프라인 음악 플레이어예요.

MP3, FLAC, WAV, ALAC, AIFF, M4A, OGG 파일을 재생하고, 앨범·아티스트·작곡가·장르·폴더 기준으로 음악을 정리할 수 있어요.

스트리밍이나 계정 없이, 직접 소장한 음악을 빠르게 탐색하고 편하게 감상하는 로컬 음악 라이브러리에 집중했어요.

Melivra는 추천 알고리즘보다 사용자의 음악 컬렉션을 더 중요하게 생각해요.

랜딩 페이지:
Melivra는 사용자가 직접 소장한 MP3, FLAC, WAV, ALAC, AIFF, M4A, OGG 파일을 정리하고 감상하는 오프라인 음악 플레이어입니다. 여러 오디오 포맷을 하나의 로컬 음악 라이브러리로 관리할 수 있습니다.

앨범, 아티스트, 작곡가, 장르, 폴더 기준으로 로컬 음악 라이브러리를 탐색하고, 통합 음악 검색으로 원하는 곡을 빠르게 찾을 수 있습니다.

스트리밍, 계정, 추천 알고리즘 없이 기기 안의 음악 컬렉션에 집중합니다. 음악 파일은 사용자의 기기에 남아 있고, 라이브러리와 감상 기록은 로컬 중심으로 관리됩니다.

긴 트랙과 반복 감상에도 맞게 이어듣기, 북마크, A-B 반복, 재생 속도 조절을 제공합니다. 앨범 커버 중심의 그리드 보기와 정보 중심의 리스트 보기를 함께 사용할 수 있습니다.

자주 묻는 질문:
질문: 음악 파일을 서버로 업로드하나요?
답변: 아니요. Melivra는 스트리밍이나 클라우드 동기화 없이 로컬 음악 파일을 기기 안에서 관리하는 플레이어입니다.

질문: 어떤 오디오 파일을 재생할 수 있나요?
답변: MP3, FLAC, WAV, ALAC, AIFF, M4A, OGG 파일 재생을 지원합니다.

질문: 오디오북이나 긴 트랙에도 사용할 수 있나요?
답변: 네. 이어듣기, 북마크, A-B 반복, 재생 속도 조절을 제공해 긴 트랙 감상에도 사용할 수 있습니다.
