Android
앱 이름:
Aligna - 파일명 정리

간단한 설명:
파일 이름을 규칙대로 한 번에 정리해요.

랜딩 부제:
일괄 파일명 정리

자세한 설명:
Aligna는 파일 이름을 규칙에 따라 깔끔하게 정리하는 일괄 파일명 변경 도구예요.

사진, 음악 파일, 문서처럼 이름을 반복해서 바꿔야 하는 파일들을 빠르게 정리할 수 있어요. 변경 전 미리보기를 확인한 뒤 적용할 수 있어 실수 없이 사용할 수 있어요.

주요 기능

• 파일 이름 앞에 날짜 추가
• 파일 이름 뒤에 날짜 추가
• 연속 번호 추가
• 공백을 언더스코어로 변환
• 금지 문자 제거
• 촬영일 기준 사진 정리
• Artist - Title 형식으로 MP3 이름 정리
• Track - Title 형식으로 MP3 이름 정리
• 적용 전 미리보기 제공
• 여러 파일 일괄 이름 변경

Aligna는 파일 탐색기나 클라우드 관리 앱이 아니에요. 파일 이름을 빠르고 일관성 있게 정리하는 데 집중한 조용한 유틸리티예요.

계정이 필요 없어요.
광고가 없어요.
클라우드를 사용하지 않아요.
모든 작업은 기기 안에서 처리돼요.

랜딩 페이지:
여러 파일 이름을 같은 규칙으로 정리하는 일괄 이름 변경 도구입니다.

Aligna는 사진, 음악, 문서처럼 이름을 반복해서 바꿔야 하는 파일을 한 번에 정리할 수 있도록 만든 앱입니다.

날짜를 앞이나 뒤에 붙이고, 연속 번호를 추가하고, 공백이나 금지 문자를 정리할 수 있습니다. 음악 파일은 아티스트와 제목 정보를 기준으로 파일 이름을 맞출 수 있습니다.

주요 작업

• 날짜 붙이기
• 연속 번호 붙이기
• 변경 전 미리보기
• 공백과 금지 문자 정리
• 촬영일 기준 사진 정리
• 음악 파일 이름 정리
• 여러 파일 일괄 이름 변경

적용하기 전에 변경될 이름을 미리 확인할 수 있어 실수를 줄일 수 있습니다.

Aligna는 파일 탐색기나 클라우드 관리 앱이 아닙니다. 파일 이름을 빠르고 일관성 있게 정리하는 작업에 집중합니다.

계정, 광고, 클라우드 저장 없이 기기 안에서 작업합니다. iOS에서는 시스템 정책에 따라 원본 파일을 직접 바꾸지 않고 새 이름으로 저장합니다.
