자세한 설명:
Aligna is a simple batch file renamer that helps you organize file names with clear, predictable rules.

Rename photos, music files, and documents without repetitive manual work. Preview every change before applying it, so you can keep file names clean and consistent.

Features

• Add dates before file names
• Add dates after file names
• Add sequential numbers
• Replace spaces with underscores
• Remove invalid characters
• Organize photos by capture date
• Rename MP3 files using Artist - Title
• Rename MP3 files using Track - Title
• Preview changes before applying
• Batch rename multiple files at once

Aligna is not a file explorer or cloud manager. It is a quiet utility focused on making file names easier to browse and manage.

No accounts.
No ads.
No cloud services.
Everything stays on your device.
