자동 포스팅 연결

자동 포스팅 연결

Bluesky 앱 패스워드와 Dev.to API key를 입력하면 로컬 env 파일에 넣을 값과 실행 명령을 만들 수 있습니다.

입력값 저장만으로는 자동 게시가 즉시 실행되지 않습니다. GitHub secrets 동기화 후 이 버튼으로 publishing workflow를 dry_run=false로 실행하세요.
복사한 env 블록을 docs/environment variables.md에 붙여넣으면 기존 자동 포스팅 스크립트가 해당 값을 읽습니다.
스토어 리뷰 연결

App Store / Play Store 리뷰 연결

Apple API 키와 Google Play 서비스 계정 JSON을 입력하면 GitHub Actions secret 설정값과 리뷰 동기화 명령을 만들 수 있습니다.

민감 값은 브라우저 안에서 GitHub 공개키로 암호화된 뒤 Actions Secrets에 직접 저장됩니다. 평문은 workflow 입력·HTML·CSV·Git에 포함되지 않습니다.
현재 필터와 일치하는 초안이 없습니다.
